# MCP Setup

## Required keys

| Server | Where to get key | Env var |
|---|---|---|
| Firecrawl | https://firecrawl.dev → Dashboard → API Keys | FIRECRAWL_API_KEY |
| SE Ranking (optional) | https://seranking.com → API | SE_RANKING_API_KEY |

## Optional additions
- Google Search Console MCP (free, open source): 
  https://github.com/ahonn/mcp-server-gsc
- DataForSEO MCP: https://github.com/dataforseo/mcp-server

## After adding keys
Restart Cursor and verify MCPs are active in Settings → MCP
